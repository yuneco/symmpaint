settings

Canvas control:

  • ↑/↓: increase/decrease pen count.
  • Space + Drag: move canvas.
  • Space + Option(Alt) + Drag: rotate canvas.
  • Space + Command(Ctrl) + Click: zoom up.
  • Space + Command(Ctrl) + Option(Alt) + Click: zoom down.

Drawing control:

  • Drag: draw line(free hand).
  • Shift + Drag: draw stright line.
  • Command(Ctrl) + Draw: Capture stroke.
  • Option(Alt) + Drag: Put captured stroke.
  • Command(Ctrl) + Z: Undo(approx. max 100 times).

キャンバス操作/設定:

  • ↑/↓: ペンの数を増やす/減らす
  • スペース + ドラッグ: キャンバス移動
  • スペース + Option(Alt) + ドラッグ: キャンバス回転
  • スペース + Command(Ctrl) + クリック: 拡大
  • スペース + Command(Ctrl) + Option(Alt) + クリック: 縮小

描画操作:

  • ドラッグ: フリーハンド
  • Shift + ドラッグ: 直線
  • Command(Ctrl) + ドラッグ: ストロークをスタンプとして記録
  • Option(Alt) + ドラッグ: 記録したスタンプを配置
  • Command(Ctrl) + Z: 取り消し（最大100回くらい）
